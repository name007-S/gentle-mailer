# 温和的邮件（Gentle Mail） - 温和、定时、AI 生成的邮件助手

一个轻量级、隐私友好的邮件调度与发送工具，帮助用户以温和、非侵入的方式发送关怀/提醒邮件。

项目灵感来源于日常生活中“想发邮件但怕打扰”的场景：用 AI 生成温暖、得体的内容 + 定时发送，避免冷冰冰的模板。

## 项目亮点
- **AI 内容生成**：集成 OpenAI/GPT，自动生成温柔、个性化的邮件开头、正文、结尾（支持随机风格）。
- **定时调度**：APScheduler 支持 Cron 表达式（如每天9:00），后台静默运行。
- **环境隔离**：dev / prod 模式区分，开发环境只预览不真发邮件。
- **FastAPI 接口**：提供 `/send-gentle-mail`（发送）和 `/preview`（预览）两个简单 API。
- **生产就绪**：已在 Vultr Debian 11 云服务器上 systemd 服务部署，长期稳定运行。
